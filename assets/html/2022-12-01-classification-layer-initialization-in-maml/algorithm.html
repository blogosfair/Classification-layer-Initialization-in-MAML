  Require: Task distribution D
  Require: η,ρ: inner loop and outer loop learning rates
  Require: Randomly initialized base-model parameters θ

2:  while not done do
3:   Sample tasks {T1,…TNbatch} from D
4:   for i = 1,2,…,Nbatch do
5:   {Ditr,Ditest}← sample from Ti
6:   θi = θ
7:   for j = 1,2,…,Nstep do
8:   θi ← θi - η∇θiLθi,Ditr
9:   end for
10:   end for
11:   Update θ ← θ - ρ∑ i=1Nbatch∇ θLθn,Ditest
12:   Set w ← 0 (the zeroing trick)
